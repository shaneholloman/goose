# goose ACP TUI

Early stage and part of goose's broader move to ACP

https://github.com/block/goose/issues/6642
https://github.com/block/goose/discussions/7309

## Running

1. Run the server `cargo run -p goose-acp --bin goose-acp-server`
2. Run the tui
    ```
    cd ui/text
    npm i
    npm run start
    ```
